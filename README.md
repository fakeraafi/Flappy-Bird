# Flappy Bird

A mobile-friendly **Flappy Bird** game built with **HTML, CSS, and JavaScript**. This project recreates the classic gameplay in the browser with a clean interface, smooth controls, and a built-in customization menu.

## Features

- Classic Flappy Bird style gameplay
- Built with pure HTML, CSS, and JavaScript
- Mobile-friendly design
- In-game customization menu
- Save customized settings
- Reset settings to default
- Clean start and game over screens

## How to Play

- Start the game from the main screen
- Tap or click to make the bird flap
- Avoid hitting pipes and the ground
- Pass through pipe gaps to score points
- Try to survive as long as possible

## Customization

The game includes a menu that lets players change available settings.  
A three-line menu button appears on the top-right when the game is not being played. Players can open the customization panel, change settings, save them, or reset everything to default.

## Technologies Used

- HTML
- CSS
- JavaScript

## Run the Project

Download or clone the project and open the HTML file in any modern browser.
